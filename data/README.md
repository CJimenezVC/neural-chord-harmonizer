# Data

See [`../DATASET.md`](../DATASET.md) for the full preparation guide.

```
data/
├── datasets/          # raw + preprocessed (gitignored; built by scripts)
│   ├── vcc2020/
│   ├── cmu_arctic/
│   └── preprocessed/  # train_features.h5, eval_features.h5, data_stats.json
├── audio_samples/     # small before/after demos (committed)
└── splits/            # train.txt, val.txt, test.txt manifests
```

Raw audio and the `datasets/` tree are **not** committed. Build them with:

```bash
./scripts/download_datasets.sh
./scripts/preprocess_data.sh
```

Split manifests (`splits/*.txt`) list utterance keys, one per line, and are
regenerated by `training/preprocess.py`.
